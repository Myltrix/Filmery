## 📚 About the Project
Bookly is a multi-page online bookstore built as the final project for a JavaScript course. It showcases core frontend skills: semantic HTML, responsive CSS, and interactive JavaScript including DOM manipulation, event handling, and form validation.

## 🧩 Pages
- Home – Welcome page with featured books
- Catalog – Browse all books with filter and search
- About – Information about Bookly and its mission
- Contact – Contact form with real-time validation
- Cart – Shopping cart with selected books and total price calculation
- Library – Full list of available books displayed as cards
- FAQ – Frequently asked questions about the service

## 📝 Forms
- Contact Form – user message form with input validation  
- Order Form – additional form demonstrating submit handling and condition checks in JavaScript

## ⚙️ JavaScript Features
- Variables & Data Types (strings, numbers, booleans)
- Conditionals & Loops (if/else, switch, for loops)
- Custom Functions (add to cart, validate form, calculate total)
- Event Handling (click, submit, input)
- DOM Manipulation (update content, toggle classes, dynamic styling)
- Form Validation (required fields, email format, checkbox validation)
